Input files format

Supported file formats: plain text files (comma-separated, tab-separated, semicolon-separated).

The first line (header) is assumed to be column names. If your file doesn't have a header, conDE will automatically name them sample-1, sample-2, etc.

First column will be assumed to be the name of the genes (or features) to be analyzed.

IMPORTANT: Remaining columns should be integers as Differential Expression packages expect a count matrix.
